# Decisiones técnicas

## Tecnologías seleccionadas

- Framework: Spring Boot
- Lenguaje: Java 17
- Tipo API: REST JSON
- Build tool: Maven

## Justificación

Spring Boot permite crear servicios REST de forma rápida y estructurada, reduciendo la configuración necesaria.

Java 17 es conocido por el equipo, lo que reduce el tiempo de aprendizaje.

Maven facilita la gestión de dependencias del proyecto.
